Aviator
A make-believe model

Aviator glasses were swiftly designed using Blender. The body is built from three curves with added geometry and exported without materials. Those are added after importing in the webpage, using threejs PB materials and one low quality environment map.

The 3D view is explorable via mouse and touch inputs, replaced materials let the user quickly customize the model's appareance, as one can do clicking on "Customize" just under this text.
